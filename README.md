# FerrySpec ⛴️

**World Adaptive Protocol for Entropy Reduction.**

基于辩证唯物主义的通用熵减协议，基于ai的边界确立了ai与人的分权。

**简述**：FerrySpec 是一个 AI 协作协议。它通过先写规划、后写代码、记录错误并持续优化的流程，确保 AI 产出符合预期。它强制同步文档与结果，防止逻辑脱节，确保所有关键决策由人类最终确认。

## 0. 渡者宣言 (The Manifesto)

### FerrySpec 是什么？

FerrySpec 不是一个工具，不是一个平台，不是一个框架。

**FerrySpec 是一个渡者。**

它渡的是：一个人的经验，到另一个人手里；这一天的对话，到明天的项目里；这一任的心法，到下一任的指尖上。

- **对抗断裂**：连接主观意图（Blueprint）与客观现实（Reality）。
- **对抗熵增**：将混乱的情绪与想法，转化为有序的产出。
- **对抗遗忘**：通过进化日志，让系统的智慧螺旋上升。

> 王国尚远，但舟已下水。

## 1. 核心公理 (Core Axioms)

本框架构建于以下四大公理之上：

### 1. 人类主权 (Human Sovereignty)

- **AI**：负责逻辑推演、信息整理、方案生成。
- **Human**：负责价值裁决、审美判断、最终拍板。

### 2. 熵减命令 (Entropy Reduction)

系统的唯一目的是将高熵（混乱/未知）转化为低熵（有序/已知）。

### 3. 双层世界观 (Dual-World View)

- **蓝图层 (Blueprint)**：凝固的意识（规划、法律、大纲）。
- **现实层 (Reality)**：流动的物质（行动、代码、产物）。
- **真理同步**：现实必须不断逼近蓝图，二者偏差即为行动动力。

### 4. 奥卡姆剃刀 (Occam's Razor)

如无必要，勿增实体。最简路径即最优路径。

## 2. 目录架构 (Directory Structure)

FerrySpec 将任意复杂事务映射为以下物质结构：

```
Project_Root/
├── .ferry/                # [内核] 存放 FerrySpec.world.md 及扩展技能
├── blueprint/             # [意识] 蓝图、大纲、法律依据、理论支撑
├── reality/               # [物质] 实际产出、草稿、代码、证据
└── archive/               # [历史] 归档任务与进化日志
```

## 3. 运行机制 (Protocol)

### 3.1 状态调度 (Universal Dispatch)

系统依据物质状态自动锁定生命周期：

| 阶段 | 状态特征 | 核心任务 |
|-----|---------|---------|
| P0 混沌期 | 蓝图缺失 | 降熵：确立核心目标，建立蓝图。 |
| P1 规划期 | 蓝图初备 | 立法：拆解任务，制定验收标准。 |
| P2 执行期 | 任务活跃 | 造物：依据蓝图执行产出，物理落盘。 |
| P3 修正期 | 出现阻滞 | 辩证：分析矛盾，修正现实或蓝图。 |
| P4 进化期 | 目标达成 | 扬弃：归档复盘，提炼新技能。 |

### 3.2 物理闸门 (Safety Gate)

所有对现实产生影响的操作必须经过分级确认：

- 🟢 **Low**：可逆操作 → 直接执行
- 🟡 **Medium**：逻辑变更 → 一句话确认
- 🔴 **High**：不可逆/高成本 → A/B 方案 + 风险评估 + 确认

### 3.3 进化逻辑 (Evolution)

任务结束时，通过辩证法三段论写入 archive/evolution.log：

1. **肯定 (Thesis)**：实现了什么物质成果？
2. **否定 (Antithesis)**：遇到了什么非预期阻滞？
3. **否定之否定 (Synthesis)**：沉淀了什么新规则或新技能？

## 4. 快速开始 (Getting Started)

**步骤 1：注入内核**

将 core/FerrySpec.world.md 的内容复制到你的 AI 助手（Cursor, ChatGPT, Claude）的 System Prompt 或 Rules 中。

**步骤 2：初始化世界**

在项目根目录创建 blueprint/ 和 reality/ 文件夹。

**步骤 3：启动对话**

根据当前状态下达指令：

- 若是一片空白 → 输入 "建立蓝图"
- 若正在攻坚 → 输入 "推进现实"
- 若陷入困境 → 输入 "辩证分析"
