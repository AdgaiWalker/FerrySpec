# 渡舟code · FerrySpec

**一个自适应提示词工程框架 —— 模块化 Skill、自动化调度、复盘归档与闭环优化。**

渡舟（FerrySpec）是一套完整的提示词生命周期管理系统。它将提示词工程化、系统化，帮助开发者构建可维护、可优化、可进化的提示词体系。

---

## 🧩 三位一体核心系统

### 1. Skill 技能系统

将提示词封装为**可复用的功能单元**。每个 Skill 包含：

- 目标定义 / 输入输出规范
- 提示词模板
- 版本、作者、依赖元数据

### 2. 调度系统

**工作流编排引擎**。根据任务需求动态选择 Skill，支持顺序、条件分支、并行、循环等执行模式。

### 3. 日志与自适应系统

**每次 Skill 执行的完整记录器与闭环优化中枢**。自动完成：

- ✅ 执行日志：输入、输出、耗时、错误、模型
- ✅ 复盘归档：结构化存储，支持追溯与分析
- ✅ 错误审视：统计 Skill 出错频率，**自动对原提示词提出增加/削减规则的建议**
- ✅ 闭环反馈：为 Skill 的持续改进提供数据驱动依据
